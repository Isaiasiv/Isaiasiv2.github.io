Angular

O Angular representa um framework fundamentado em JavaScript empregado na elaboração de Aplicações de Página Única (SPA). Para além do âmbito web, as iniciativas edificadas com este framework ajustam-se igualmente ao ambiente móvel e de secretária. Surgido em 2016, o Angular faz uso do Typescript para a elaboração de interfaces, de forma descomplicada e eficiente.

Angular

Versões


O AngularJS representa a iteração anterior ao Angular. Sua estreia ocorreu em 2009, sendo a versão 1.x do framework. Em 2016, emergiu uma segunda iteração inteiramente remodelada. No início, a segunda versão da tecnologia recebeu o nome de Angular 2, posteriormente simplificado para Angular. Através dessa reestruturação, o Angular incorporou novas funcionalidades, incluindo a integração do Typescript, bem como a adoção de técnicas e práticas avançadas, alinhadas aos padrões da web contemporânea.

O funcionamento do Angular é baseado na organização por meio de módulos. Estes módulos configuram-se como agrupamentos de código, empregados para a construção da sua arquitetura. Além das unidades pré-existentes, o framework também faculta a inclusão de módulos externos, provenientes de terceiros, os quais expandem as capacidades do Angular e viabilizam a concepção de projetos mais intrincados. Quando se trata da codificação, essa tecnologia se vale da linguagem de programação Typescript. Conforme ocorre a compilação do código Typescript, é gerado um código JavaScript, capacitando a execução da aplicação nos navegadores sem enfrentar entraves de compatibilidade.

Instalando o Angular Cli

node

O ponto de partida inicial ao utilizar o Angular CLI envolve a instalação do mesmo, requisitando a presença do Node.js já instalado. Se porventura ainda não o possui, baixa em Node js e obtenha a versão mais recente compatível com o seu sistema operacional.

baixando angular

Apois a instalação do node js Temos à nossa disposição o utilitário npm, necessário para instalar o Angular CLI. Nesse ponto, será preciso abrir o terminal do seu sistema operacional e digitar a instrução:
npm install -g @angular/cli

Essa diretriz, em geral, exige um período considerável para sua conclusão, uma vez que implica na obtenção via internet de múltiplas bibliotecas do Node.js. Uma vez que a instalação esteja finalizada, o Angular CLI se torna acessível por meio do comando "ng", um utilitário destinado à execução a partir da linha de comando, e que pode ser acompanhado por diversos argumentos. É dessa maneira que desencadeamos a criação de um novo projeto em Angular, geramos componentes, efetuamos a compilação do projeto, além de outras alternativas.
